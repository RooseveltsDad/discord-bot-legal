# Privacy Policy

Last updated: 04 January 2026

This Discord bot respects user privacy.

## Data Collection
The bot may collect Discord user IDs, server IDs, moderation metadata, and activity data via MEE6.

## Data Usage
Data is used only for moderation, automation, and server management features.

## Data Sharing
No data is shared with third parties.

## Data Storage
Data is stored securely and only as long as necessary.

## Data Removal
Data can be removed by removing the bot from the server or contacting the bot owner.
